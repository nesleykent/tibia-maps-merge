Tibia Maps Merge

Tibia's map changes with every game update, and tibiamaps.io keeps its community markers updated to match. This merges your personal markers with the latest community ones -- so you get both, with your own winning if you've marked the same spot differently.

How it works

  1. We fetch the latest community markers straight from tibiamaps.io -- the same file it uses for its own map installer, so it's always current with the latest Tibia update.
  2. You upload your own markers -- the minimapmarkers.bin file from your Tibia client's minimap folder (or a markers.json, if you already have one).
  3. We merge them, entirely in your browser -- markers are matched by their map coordinates. If you and the community both marked the same spot, yours wins.
  4. You get a new minimapmarkers.bin back. Drop it into that same minimap folder (replacing the old one) and restart your Tibia client.
Where do I find my minimapmarkers.bin?
Windows
%LOCALAPPDATA%\Tibia\packages\Tibia\minimap
macOS
~/Library/Application Support/CipSoft GmbH/Tibia/packages/Tibia.app/Contents/Resources/minimap
Linux
~/.local/share/CipSoft GmbH/Tibia/packages/Tibia/minimap

From tibiamaps.io's installation guides -- check there if your setup differs (e.g. a non-default install location, or the Test Server client).
