Tibia Maps Merge

Tibia's map changes with every game update, and tibiamaps.io keeps its community markers updated to match. This merges your personal markers with the latest community ones -- so you get both, with your own winning if you've marked the same spot differently. Everything runs in your browser; nothing is uploaded anywhere.
