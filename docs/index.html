Tibia Maps Merge

Convert raw minimap exports to PNG + JSON, and merge them -- entirely in your browser. Nothing is uploaded anywhere.

Pick your Tibia client's minimap export folder (the one full of Minimap_Color_*.png files) to produce PNG floor maps + JSON markers, downloaded as a single .zip.

Merge two or more sources -- raw exports or already-converted data folders, any mix. Markers are unioned by coordinate; explored map/path tiles are unioned so gaps in one source get filled from another. Order matters only when sources disagree on the same tile or marker -- later sources win.

Merge any number of marker files directly -- .bin snapshots from the Tibia client and/or already-converted markers.json files. Files that can't be parsed (e.g. very old marker formats) are skipped with a warning instead of failing the whole merge.

End in .json or .bin to choose the output format.

Inspect a raw export or data folder -- bounds, floors, tile counts, marker count.
